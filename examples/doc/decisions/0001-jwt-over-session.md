# ADR 0001 — 세션 대신 JWT 사용

- **상태**: 채택 (2026-07-14)
- **맥락**: 향후 MSA 확장 예정. 서버 간 세션 공유(스티키 세션·중앙 세션 저장소) 부담을 피하고 싶다.
- **결정**: 인증은 JWT(무상태) 액세스 토큰 + 리프레시 토큰 조합.
  - 액세스 토큰 15분(짧게), 리프레시 토큰 14일.
- **트레이드오프**: 토큰 즉시 폐기가 어렵다 → 짧은 액세스 만료 + 리프레시 회전(rotation) + 블랙리스트로 보완.
- **대안(기각)**: 서버 세션 — 수평 확장 시 공유 비용, MSA와 부적합.
- **영향**: `user` 도메인 계약(`RefreshEndpoint`·`U003`), 인증 필터 전반.
