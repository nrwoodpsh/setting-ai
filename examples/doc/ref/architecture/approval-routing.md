# 아키텍처 사실 — 결재선 결정 (approval routing)

> `/spike` 검증(2026-07-16)에서 승격된 **확정 제약**. 근거: ADR [`0002`](../../decisions/0002-approval-line-rule-engine.md). 이후 `approval` 설계·구현은 이 사실을 정본으로 참조한다.

- **결재선 결정 = 규칙엔진 + 예외 오버라이드 테이블.** ML 기반 예측은 감사 설명가능성 요건으로 **채택 불가**.
- **자동 결정은 강제가 아니다.** 자동 96% 커버, 나머지는 기안자가 제출 전 수정 가능(감사 + 규칙 밖 케이스).
- **예외 규칙은 데이터**(코드 아님) — 조직 개편 시 무배포 변경.
- **금액 임계는 설정값** — 하드코딩 금지 (검증 시점 기준값: 500만 원, 변동 가능).
